This folder serves as the root for your static assets.

Upload your files here:
- image1.jpg (or image1.png)
- image2.jpg (or image2.png)
- image3.jpg (or image3.png)
- video.mp4

Note: The web app automatically looks for `.jpg` files first. If you upload a `.png` without changing the code, it will still show the placeholder. If you want to use PNGs, you can modify the `src` paths in `src/screens/Dashboard.tsx` to point to `.png`.
